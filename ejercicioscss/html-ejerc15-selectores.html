Tipos de selectores que entran en el examen

Clase (class=””)

Esto sirve para poder cambiar el estilo de un conjunto de texto concreto, es decir, sirve para aplicar estilos a todo texto que incluyas dentro de la clase que crees. Por ejemplo,voy a aplicar una clase llamada "rojo" y voy a aplicarla a este párrafo para cambiar el color. De esta manera solo se cambiara el color de el texto que este dentro de esta clase.

Identificador (id=””)

Esto sirve para lo mismo que la Clase, pero la diferencia es que en este caso, este identificador solo sirve para cambiar el estilo de una única cosa. Por ejemplo voy a aplicar un id llamado "A" a este parrafo para poner el color azul.

Todos los elementos (*)

Esto selector sirve para aplicar algún estilo a todo el contenido de la página. Por ejemplo voy a aplicar este selector para cambiar el tipo de letra.

Varios elementos ( p, em, h1, h2)

El selector p sirve para aplicar un estilo a todos los párrafos de la página. Por ejemplo,voy a aplicar este selcetor para poner todos los párrafos alineados en el centro.

El selector em sirve para poder poner en letra cursiva cualquier palabra o texto que queramos y ademas para poder aplicarle un estilo solo a esa palabra. Por ejemplo voy a aplicar este selector a la siguiente palabra que además le aplicare color morado: España

El selector h1 sirve para aplicar un estilo a todos los h1 (titulos). Por ejemplo, voy a poner todos el fondo de todos los h1 de color verde.

El selector h2 sirve para lo mismo que el selector h1 pero en este caso cambiara el estilo de todos los h2. Por ejemplo, voy a poner el fondo de todos los h2 de color naranja.

Elementos contenidos en otros elementos (ol li)

El selector "ol" sirve para ordenar una lista de elemntos de tal forma que aparezcan marcados con numeros o letras. El selector "li" sirve para poder crear la lista (ordenada o no ordenada) y que los elementos aparezacan uno debajo de otro y que puedan tener un estilo prpio. Se utiliza dentro de las listas "ol" o "ul". Por ejemplo voy a crear una lista ordenada de tres elementos:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
